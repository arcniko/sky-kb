# Content Directory

## Atlas (`content/atlas/`)
Sky Atlas governance documentation — one markdown file per scope, synced from sky-atlas.io.

Documents use dot-separated formal IDs (e.g. `A.1.2.3`) and type labels in brackets:
`[Scope]`, `[Article]`, `[Section]`, `[Core]`, `[Type Specification]`, `[Subtype]`.

## Laniakea Docs (`content/laniakea-docs/`)
Current core work on the Sky Protocol. Prioritize this for protocol-level questions.

## MCD Docs (`content/mcd-docs-content/`)
Maker Protocol (MCD) technical documentation. Older but foundational.
